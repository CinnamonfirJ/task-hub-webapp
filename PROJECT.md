# Project Overview

## 📖 Description
Task Hub Web App is a modern, full‑stack platform that connects task posters with skilled taskers. It supports task creation, bidding, real‑time chat, identity verification via Didit, and a wallet system handling both fiat and Stellar cryptocurrency with escrow support.

## 🛠 Tech Stack
- **Framework**: Next.js (App Router) – server‑rendered React
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4 + Shadcn UI components
- **State Management & Data Fetching**: React Query (`@tanstack/react-query`)
- **Authentication**: Custom auth + Google OAuth (`@react-oauth/google`)
- **Forms & Validation**: React Hook Form + Zod
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Identity Verification**: Didit integration
- **Payments**: Fiat & Stellar crypto via wallet service

## 📂 Project Structure
```
app/                # Next.js App Router (pages, layouts, route groups)
components/         # UI components (Shadcn UI primitives + feature components)
hooks/              # React Query custom hooks for each domain entity
lib/api/            # Pure service‑layer functions making fetch calls
types/              # TypeScript interfaces for API payloads & models
public/             # Static assets (images, fonts, etc.)
utils/              # Shared utility functions (e.g., className helper)
```

## 🚀 Getting Started
### Prerequisites
- Node.js ≥ 18.17
- npm or yarn

### Installation
1. Clone the repository
```bash
git clone <repository-url>
cd task-hub-webapp
```
2. Install dependencies
```bash
npm install
```
3. Create an environment file (`.env.development` or `.env`) with the required variables (refer to the README for the full list).

### Development Server
```bash
npm run dev
```
Visit `http://localhost:3000`.
